# Ecosystem
An ecosystem simulator inspired by [Coding Adventure: Simulating an Ecosystem](https://youtu.be/r_It_X7v-1E)

## Installation
You can use cmake to build this library.
```bash
mkdir build && cd build
cmake ..
make -j8
```

## Usage
```bash
ecosystem
```
